mgwt.intersect
==============

Basic web application with offline capabilities

Environment:
-----------

1. Eclipse (with GWT plug-in)
2. GWT 2.5.1
3. Apache Maven
 
Step by step:
------------

1. Clone the project
2. Go into the project forlder and run the followings commands   
	mvn eclipse:eclipse   
	mvn war:exploded   
	mvn clean package   

3. Import the project into eclipse.
